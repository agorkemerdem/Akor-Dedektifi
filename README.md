# Akor Dedektifi

Akor Dedektifi, Chrome Music Lab uygulamalarından esinlenerek geliştirilmiş web tabanlı bir kulak eğitimi uygulamasıdır.

## Özellikler

- Majör, Minör, Sus2, Sus4, Dominant7, Maj7 ve Min7 akorlarını tanıma
- Kulak eğitimi oyun modu
- Eğitim modu
- Piyano üzerinde akor gösterimi
- Gitar üzerinde akor diyagramı gösterimi
- Puanlama sistemi
- İstatistik ekranı
- Responsive tasarım

## Teknolojiler

- HTML5
- CSS3
- JavaScript
- Web Audio API

## Amaç

Kullanıcının duyduğu akorun türünü tahmin etmesini sağlayarak kulak eğitimi becerilerini geliştirmek ve akorların teorik yapısını öğretmek.

## Geliştirme Süreci

Bu proje, yapay zeka destekli geliştirme araçları kullanılarak eğitim amaçlı hazırlanmıştır.
